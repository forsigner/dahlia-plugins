# dahlia-antd-pro

## Installation

```sh
yarn add dahlia-antd dahlia-antd-pro
```

## Usage

```js
import antd from 'dahlia-antd'
import antdPro from 'dahlia-antd-pro'

export default {
  plugins: [antd(), antdPro()],
}
```
